# 排程系統快速上手指南

## 🚀 新功能一覽

你的排程頁面現在支援：

1. **🔍 可縮放時間軸** - 從全局到分鐘級精度
2. **🎯 拖動輔助線** - 即時顯示時間和工時
3. **🗺️ Minimap 導航** - 快速跳轉到任何時間段
4. **📊 智能卡片** - 根據縮放自動調整顯示密度
5. **⚡ 快捷操作** - Shift/Ctrl + 滾輪快速操作

---

## 📖 5 分鐘上手教學

### 1️⃣ 調整縮放比例

**使用滑桿**：
```
工具列 → 縮放 → 拖動滑桿 [0.5x ──●─── 10x]
```

**使用快捷鍵**：
```
Ctrl/Cmd + 滾輪上下 → 放大/縮小
```

**重置視圖**：
```
點擊「重置」按鈕 → 回到 1x
```

### 2️⃣ 水平捲動

```
Shift + 滾輪上下 → 左右捲動時間軸
```

或使用底部滾動條

### 3️⃣ 拖動工單

1. **點擊並按住**卡片
2. 觀察**藍色對齊線**和**浮動 tooltip**
3. 移動到目標機台和時間
4. **放開滑鼠**完成移動

> 💡 提示：紅色虛線表示衝突（停機時段），無法放置

### 4️⃣ 使用 Minimap

底部縮略圖顯示：
- 所有工單的相對位置
- 當前可見範圍（藍色框）

**快速跳轉**：
```
點擊 Minimap 任意位置 → 立即跳到該時間段
```

### 5️⃣ 查看不同時間精度

| 場景 | 建議 Zoom | 刻度顯示 |
|-----|-----------|---------|
| 全天排程概覽 | 0.5x - 1x | 每小時 |
| 半天排程調整 | 1.5x - 3x | 每 30 分 |
| 精細時間規劃 | 3x - 6x | 每 10 分 |
| 分鐘級調度 | 6x - 10x | 每 1 分 |

## ⌨️ 快捷鍵清單

| 操作 | 快捷鍵 | 說明 |
|-----|--------|------|
| 放大 | `Ctrl/Cmd + 滾輪上` | 增加時間軸縮放 |

| 向左捲 | `Shift + 滾輪下` | 時間軸向左移動 |

## 🎯 常見使用場景

### 場景 A：查看今日全部排程

1. 設定 zoom = **0.5x** 或 **1x**
2. 一次看到所有 12 小時
3. 快速掃描機台負載

### 場景 B：調整工單時間

1. 放大到 **3x - 6x**
2. 拖動工單到精確時間
3. 觀察 tooltip 確認開始/結束時間

### 場景 C：插入急單

1. 找到空檔（可用 Minimap 快速定位）
2. 放大查看精確時間
3. 拖動工單到該時段
4. 確認無衝突後放置

### 場景 D：檢查停機影響

1. 縮小到全局視圖
2. 觀察紅色停機區塊
3. 確認無工單衝突

## 💡 進階技巧

### 技巧 1：精確到分鐘的排程

```
1. Zoom 到 10x
2. 時間刻度會顯示每分鐘
3. 拖動時 tooltip 會顯示精確時間 (HH:MM)
```

### 技巧 2：快速對比兩個時間段

```
1. 使用 Minimap 快速跳轉
2. 點擊早上 → 查看排程
3. 點擊下午 → 查看排程
```

### 技巧 3：避免拖動失誤

```
- 拖動時觀察藍色對齊線
- Tooltip 會即時計算新時間
- 紅色虛線 = 無法放置
- 放開滑鼠前確認 tooltip 顯示正確
```

## 🔧 故障排除

### Q1: 卡片顯示太小看不清？

**A**: 放大縮放比例 (zoom > 2x)，卡片會自動切換到精簡模式但更容易定位

### Q2: 時間軸太長找不到目標時間？

**A**: 使用底部 Minimap 點擊跳轉，或使用 Shift + 滾輪快速捲動

### Q3: 拖動時卡片跑到奇怪位置？

**A**: 
- 確認沒有衝突（紅色虛線）
- 確認時間在 08:00-20:00 範圍內
- 確認 tooltip 顯示的時間正確

### Q4: 縮放後畫面卡頓？

**A**: 
- 關閉其他瀏覽器分頁
- 使用篩選功能減少顯示的工單數量
- 避免極端縮放 (< 0.5x 或 > 10x)
